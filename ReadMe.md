# 🛡️ Brute-Force Login Attack Detector
> **Applying Machine Learning to Cybersecurity** — A lightweight ML model that detects brute-force login attacks in real time using behavioral features, wrapped in an interactive Streamlit web app.

##  Overview
This project explores the intersection of **AI and cybersecurity** by training a Random Forest classifier to distinguish between normal user login behavior and brute-force attack patterns purely from behavioral metadata (no passwords, no raw logs required).
Built as a hands-on mini project to practice applying ML to real-world security problems.

##  Problem Statement
Brute-force attacks account for a significant share of unauthorized access incidents. Traditional rule-based detection (e.g., "block after 5 failed attempts") is rigid and easily bypassed. This project demonstrates how a simple ML model can learn **patterns** that are harder to fool.

##  How It Works

### Features Used
| Feature | Description |
|---|---|
| `login_attempts` | Total number of login attempts in a session |
| `failed_attempts` | Number of failed login attempts |
| `failed_ratio` | Ratio of failed to total attempts (auto-computed) |
| `avg_time_between` | Average seconds between consecutive attempts |
| `ip_change_frequency` | How often the source IP changes (0 = never, 1 = always) |

### Model
- **Algorithm:** `RandomForestClassifier` (scikit-learn)
- **Training Data:** 2,000 synthetically generated sessions with realistic normal vs. attack distributions
- **Accuracy:** 100% on the test split
- **Output:** Binary label — `0 = Normal Activity`, `1 = Brute-Force Attack`


##  Demo

The Streamlit app provides:
- Interactive sliders for all input features
- Real-time prediction with color-coded result (✅ Normal / 🚨 Attack)
- Attack probability gauge (%)

##  Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/khaoulaettijani-iacs/cyber_ml_detector.git
cd cyber_ml_detector
```

### 2. Install dependencies

```bash
pip install scikit-learn pandas numpy streamlit matplotlib
```

### 3. Train the model

```bash
python model.py
```

This generates `model.pkl` and prints accuracy + confusion matrix to the terminal.

### 4. Launch the web app

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.
